 Basic Chatbot:

  A simple rule-based chatbot developed using Python that responds to predefined user inputs.

 Features:
     
  Responds to greetings and common questions
  Continuous conversation using loops
  Exit option using "bye"
  Easy to customize with new responses

 Technologies Used:
   Python

 How to Run:
   Download the project files.
   Open the terminal in the project folder.
   Run:
      python chatbot.py
